## Installation

1. Download and extract `Minori-Codex-Pet.zip`.
2. Copy the complete `minori` folder into `~/.codex/pets/`.
3. In Codex, open **Settings > Pets**.
4. Click **Refresh**, select **Minori**, and click **Select**.
5. Click **Wake Pet** if the floating pet is hidden.

The installed files should be:

```text
~/.codex/pets/minori/pet.json
~/.codex/pets/minori/spritesheet.webp
```

## Package Contents

- `minori/pet.json`: Codex pet manifest using `spriteVersionNumber: 2`
- `minori/spritesheet.webp`: validated 8x11 animated sprite atlas
- `minori/INSTALL.txt`: plain-text installation guide
